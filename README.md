# Meditation Timer

A free, private meditation timer with ambient sounds.

- 100% runs on your device
- No sign-ups required
- No data collection

## Features

- Duration selection: 5, 10, 15, 20, 30 minutes
- Ambient sounds: Rain, Forest, Ocean, Wind, Tibetan Bells, Silence
- Volume control
- Session counter
- Responsive design

## Deploy

Connect to Vercel and deploy automatically from GitHub.
